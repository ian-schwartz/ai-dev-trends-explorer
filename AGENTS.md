# AI Dev Trends Explorer

This project is an interactive web application that explores the modern AI development ecosystem.

The goal is to provide developers with a clear overview of:

- AI-native IDEs
- Coding agents
- Prompt-to-app tools
- Modern AI development workflows

The project should feel like a polished developer dashboard rather than a blog.

## Stack

Next.js  
TypeScript  
Tailwind CSS  
Framer Motion  

## Project Structure

src/app → pages  
src/components → UI components  
src/data → static datasets  
src/lib → utility functions  

## Core Principles

- Keep architecture simple
- Avoid unnecessary complexity
- Prefer reusable components
- Use strongly typed data
- Maintain clean UI hierarchy
- Ensure responsive layout

## Implementation Guidelines

- Changes should remain scoped
- Do not rewrite unrelated code
- Maintain consistent component structure
- Avoid introducing backend services

## Quality Bar

The application should feel:

- modern
- polished
- structured
- useful for developers

UI should emphasize clarity and readability.
